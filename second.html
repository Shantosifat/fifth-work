getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(),getAttribute(), querySelectorAll(),childNodes

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML returns HTML tags with content. InnerText returns only text content inside the HTML tag with the style in it. TextContent retrieves or sets all text, ignoring styles and HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element instead of individual child elements. This works because events bubble up from the target element to its ancestors, allowing the parent to handle events for multiple children.

Question-4: What is event bubbling in the DOM?

Event bubbling is where an event starts from the target element and "bubbles" up to its parent elements, then to its grandparent elements, and so on, all the way up to the body tag.It can stop between anywhere if we use event.stopPropagation inside the event handler.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an Element:Use document.createElement('elementType') to create a new element.
Add the Element: Use parentElement.appendChild(newElement) to add the element to the DOM.
Remove the Element:Use element.remove() to remove an element directly.
